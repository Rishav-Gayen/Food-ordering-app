# 🍔 Scrimba Food Ordering App

This is a simple **Food Ordering App** built as part of the [Scrimba Frontend Developer Career Path](https://scrimba.com/learn/frontend) course. The app was developed using **HTML**, **CSS**, and **Vanilla JavaScript**, without any frameworks or libraries.

It allows users to browse a menu, add items to a cart, view a live order summary, and complete a mock checkout flow — giving a real-world feel for a food ordering interface.

---

## 🚀 Features

- 🧾 Dynamic menu rendered from JavaScript  
- ➕ Add & remove food items from order  
- 💰 Real-time order total calculation  
- 🧍 Mock checkout modal with name input  
- ✅ Order confirmation screen after checkout  
- 🎨 Clean, responsive UI styled with pure CSS  

---

## 🛠️ Tech Stack

- **HTML5** – For structuring the app  
- **CSS3** – For layout, styling, and responsiveness  
- **Vanilla JavaScript (ES6+)** – For DOM manipulation and app logic
